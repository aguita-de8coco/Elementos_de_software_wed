Cuestionario Resuelto
1. Consulte y explique que significa a nivel de uso del internet:

a. Servidor DNS: Es como la "agenda telefónica" de internet. Traduce nombres de dominio (como google.com) en direcciones IP numéricas para que el navegador sepa a dónde ir.

b. IP: Es la dirección única que identifica a un dispositivo en internet o en una red local (ej. 192.168.1.1).

c. Protocolo: Es un conjunto de reglas que permiten que dos dispositivos se comuniquen entre sí (ej. HTTP para páginas web).

d. Nombre de dominio: Es el nombre amigable que escribimos en el navegador para entrar a un sitio (ej. facebook.com).

e. Directorio y nombre de archivo: Es la ruta específica dentro de un servidor que indica dónde está guardado un recurso (ej. /imagenes/foto.jpg).

2. Roles en el desarrollo web:

a. Diseñador UI/UX: Se encarga de cómo se ve el sitio (UI) y de que sea fácil y agradable de usar para el usuario (UX).

b. Cloud Manager: Gestiona los servidores y servicios en la nube (como AWS o Azure) donde se aloja la aplicación.

c. Administrador de base de datos: Organiza, almacena y protege los datos de la aplicación para que estén disponibles y seguros.

d. Project Manager: Coordina al equipo, los tiempos y los recursos para asegurar que el proyecto se entregue correctamente.

3. ¿Qué significa que un lenguaje sea turing completo?

Significa que el lenguaje tiene la capacidad de resolver cualquier problema algorítmico o matemático, siempre que tenga suficiente tiempo y memoria (puede hacer bucles, condiciones y manipular datos).

4. ¿Por qué CSS y HTML no son lenguajes de programación?

Porque no tienen lógica de programación propia: no pueden realizar cálculos complejos, no tienen bucles (for, while) ni toma de decisiones condicionales de forma nativa. HTML es de marcado (estructura) y CSS es de estilos (diseño).

5. Defina con sus palabras qué es una etiqueta en HTML:

Es un código especial que encierra un contenido para decirle al navegador qué tipo de elemento es (un título, un párrafo, un enlace, etc.) y cómo debe mostrarlo.
